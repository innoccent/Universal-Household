## 关于rem
* 页面缩放比例设置为0.5(前台html已设置为0.5);
* 设计尺寸设置为375(rem中以设置)
* 具体计算方式实例:
```css
        /** px写法 **/
       div{
            width:  200px;
            height: 200px;
       } 
       /** rem写法 **/
       div{
            width: 1rem;
            height: 1rem;
       }
       /** 正常ip6下设计换算比例为200，望周知 **/ 
```
